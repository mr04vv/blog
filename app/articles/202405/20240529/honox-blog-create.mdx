## MDXがすごい

MDXは簡単に言うと「MarkdownにJSXが書けるもの」なんですが、これが普段JSXを書いてる身からすると超便利です。

[Storybook](https://storybook.js.org/)で見たことはあったものの自分で使う機会がなかったので初めてちゃんと使ってみました。

MDXは独自コンポーネントも使えるので、画像を表示したいときはこんな感じで書けちゃいます。
```tsx
<Image src="./latte.jpg" caption="今日のらてさん" alt="今日のらてさん"/>
```

JSXとかその前にMarkdownでブログ書けるのめっちゃ便利です。

## シンタックスハイライトがすごい

今回は[Rehype Pretty Code](https://rehype-pretty.pages.dev/)を使ってシンタックスハイライトを実現しました。

こんな感じで行数表示とか行ハイライトとかいい感じにやってくれます。
```tsx showLineNumbers{} title="sample.tsx" {5,8-9} /setFloating/#s /refs/#v /adapter/
import { useFloating } from "@floating-ui/react";
 
function MyComponent() {
  const { refs, floatingStyles } = useFloating();
 
  return (
    <>
      <div ref={refs.setReference} />
      <div ref={refs.setFloating} style={floatingStyles} />
    </>
  );
}
```
